Flight Ninja

UI design + CMS + web app

Flight Ninja is a web app designed for teachers at an Aviation School. It was customized for the school's needs, with the options of scheduling classes and flight simulations, creating courses, grading lessons and many more requirements for students.

The Aviation School that contacted us already has a system in place; this was designed poorly and needed many improvements. They got in touch to request a more user-friendly design. They provided us with the logo and the access to their existing web app.

Upon analyzing everything that needed to be integrated into the UI, I began evaluating how each aspect would work. After a long meeting with the client explaining all that has to do with an Aviation School, I designed an intuitive and actionable system to help users meet their goals.
